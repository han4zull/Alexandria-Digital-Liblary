# Alexandria-Digital-Liblary
Alexandria Digital Library adalah aplikasi perpustakaan digital berbasis web yang memudahkan pengguna dalam mengakses, meminjam, dan mengelola koleksi buku secara online. Sistem ini dirancang untuk membantu admin dan petugas dalam mengelola data buku, transaksi peminjaman, pengembalian, serta laporan secara efisien dan terstruktur.
